MarkDowm的快速上手指南

· 2023-04-22 · # 其他

1.标题使用方法

一级标题前加#

二级标题前加##

三级标题用前加###

四级标题前加####

五级标题前加#####

2.删除线使用方法

删除线：文字左右两边各加两个~~

例：~~删除线~~

3.斜体

_斜体_：文字左右两边各加*

例：*斜体*

4.加粗

加粗：文字两边各加**

例：**加粗*

5.斜体+加粗

**斜体+加粗**：文字两边各加***

例：***斜体+加粗***

6.下划线

下划线

例：<u>下划线</u>

7.高亮（需勾选扩展语法）

高亮：文字两边各加==

例：==高亮==

8.下标（需勾选扩展语法）

H2O：在下标左右加~

例：H~2~O

9.上标（需勾选扩展语法）

X2:在上标左右加^

例：X^2^

10.引用

引用：在引用的内容前加>

例：>引用

嵌套引用：在嵌套引用的内容前加>>

例：>>嵌套引用

11.列表

(1)无序列表

  • 可以用 （一个''*''加一个''空格''） 标记

  • 也可以用 （一个''+''加一个''空格''） 标记

  • 还可以用（一个''-''加一个''空格''） 标记

(2)有序列表

  1. ''数字''加一个 “.” 加一个''空格''

  2. ''数字''加一个 “.” 加一个''空格''

12.代码

(1)代码块

```语言名称

例：

#include<iostream>
using namespace std;
 int main()
 {
     cout<<"Hello World !"<<endl;
     system("pause);
     return 0;
 }

(2)行内代码

markdowm:代码行左右两边加一个"`"

例：`markdowm`

13.分割线

上面这是分隔线：

“***” 加 “一个空格” 或 “---” 加 “一个空格”

14.跳转

（1）外部跳转—超链接

博客园（按住‘’Ctrl“点击才能跳转到相应网址）

例：[提示文字](网址地址)
   [博客园](https://home.cnblogs.com/blog/)

（2）内部跳转—文本内部跳转

跳转到10.引用（按住‘’Ctrl“点击才能跳转到相应内容）

例：[提示内容](#想要跳转到本文本内的内容)

（3）自动链接

https://home.cnblogs.com/blog/

例：<https://home.cnblogs.com/blog/>
   <这里面的内容会自动变成超链接>
