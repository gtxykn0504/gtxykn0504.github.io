关于blog/网站的简易搭建

2023-10-03 · 7 min read
网站与博客

一.What？

”何为blog？“

很早之前，我就已经很想聊一聊关于blog这件事以及搭建网站的过程。但是，网站也有许多的教材是关于搭建网站的，但是文章内容大多数是比较专业的，或者内容不详细，对与普通人可能会有阅读困难，所以本篇文章就是对小白或者不爱折腾的人搭建网站的教程。如果你是一个对网站有“向往之心”的人，你可以看看这篇文章，或许对你有帮助。

我其实搞这个网站有一点原因是因为装逼 更好的分享文件与观点，也是有一个可以随心所欲使用的个性化email。在搭建这个网站之前希望你可以先好好的想想你的网站的作用，这很重要，不同的需求有不同的表达形式。

二.How？

2.1购买域名

Tips：域名与DNS解析内容见 信息技术-域名与DNS解析

1.选择域名注册商

选择正规的域名注册商，进行域名购买，比如腾讯云，阿里云，华为云等服务。

防诈小提示：如果您心爱的域名已被他人注册，最好不要私下与域名主进行交易。

2. 以阿里云为例子，我们要先注册阿里云账户，然后经过实名认证。

注意：强烈建议个人账号不要用于企业业务的购买！一定分清个人认证和企业认证：完成企业实名认证的账号归属为企业。完成个人实名认证的账号归属为个人。

3.在查询框中输入您想要的域名，单击查域名。

3. 对于查询结果为未注册的域名，单击加入清单。

4. 在域名清单中确认已添加的域名，单击去结算。

5. 在确认订单页，选择域名的购买年限和持有者类型。

6. 在购买途中可能需要选择域名信息模板。

  • 如果您的域名持有者信息模板已创建完成，您可以选择一个域名信息模板。

  • 如果您未提前创建好域名持有者信息模板，您可以单击创建新的信息模板，并如实填写域名持有者信息，并且进行认证。完成后回到本页面，选择您创建的信息模板。

注意：域名注册必须使用已完成实名认证的信息模板；未实名认证域名不能续费、将被暂停解析。

7. （可选）许多提供商针对不同的域名后缀，会不定期提供一些优惠服务，例如优惠口令等。您可根据界面显示的内容选择是否使用优惠服务。

8. 确认服务条款后，勾选我已阅读，理解并接受《域名在线服务条款》，单击立即购买。

9. 选择支付方式后，单击立即支付。

说明：域名注册须即时报送注册局系统，只有最终完成付款才会向注册局发送指令，下单后请尽快结算。付款方式有多种，如账号余额、支付宝、网银等，您可根据实际情况自行选择。

10.完成支付后，域名注册订单会短暂的为“处理中”状态，待订单状态变为“成功”，域名即注册成功。

11. 域名注册成功后还需要提交与域名注册信息相同的资料进行实名认证审核，部分特殊域名（例如.cn）还需进行资质审核。通过审核后域名才能正常使用。

2.2选择托管平台

搭建网站正常需要购买服务器等相关资源，但为方便，我们可以选择托管网站。托管网站会提供服务器和其他资源，我们这边以Gridea为例子说明。除此之外，也有许多其他的托管平台可以进行选择，例如谷歌的Blogger，按需选择即可。

2.3托管平台的“初始化调整”

由于不同的托管网站会提供不同形式的后台，但基本上能够调整的内容都差不多，例如网站的头像，名称板块。根据，不同托管网站的说明使用即可调整。

以Gridea为例，它的后台可以分为一下部分，下面我们就来了解下。

1.多网站调整：选择进行操作的不同网站，Gridea支持一个账户创建互相独立的3个网站，我们可以分别对调整控制。

2.侧边栏：功能选择

  • 帖子：所有的帖子都会集中到页面“帖子列表”中，你可使用下方“设计”这一侧边栏将这一元素放至于网站。无需多次同步，帖子将会自动更新至你的网站。

  • 页面：页面是独立的。不会出现在“帖子列表”中。帖子是会出现在列表中的。（你可以把它作为“关于”页，以此介绍自己）

  • 标签：你可以将帖子进行分类，打上不同的标签。

  • 设计：使用该功能可以定制你的网页主页以及帖子字体等，是一个不错的自定义工具。

  • 设置：在该侧边栏中可以更改你网站的名称，头像，域名，网站导航（它会固定显示在你网站的顶部）或者更多高级功能。

  • 访问统计：统计你的读者。所在位置使用的设备以及数量。

3.新帖子：点击此处可以新建一个帖子（有点像新建一个word，但是它是使用markdown进行编辑的）

Tips： Markdown是一个标记语言。你可以通过 Markdown 基本语法 | Markdown 官方教程 进行学习，它会提高你编写的速度。

4.顶部栏：Gridea还提供了许多其他的功能，你可以慢慢进行探索，让它成为你知识库搭建的一部分。

5.个人：在这里，你可以查看你的个人信息以及付费情况等相关资料。

2.4DNS解析

Tips：域名与DNS解析内容见 信息技术-域名与DNS解析

看到这里，你或许会疑惑：“诶，我的域名到底有什么用呢？”别急，让我们把域名“绑”在你的页面上。这时我们就需要进行DNS解析。

我们还是以Gridea为例，我们可以在设置中看到“托管设置”这一部分。

接下来请打开您的域名控制面板。选择“DNS解析记录”，添加一条记录，根据托管设置中的内容**（不同托管平台所填写的内容不同，请一定注意）**进行填写记录，完成后点击“保存”

最后一步！回到Gridea设置，将你所购买的域名栏填写到“托管设置”（“域名”下方的填写栏），“魔法”大功告成！

提示：不同的托管平台可能方式不同，但是原理都差不多，你可以根据您的托管平台进行调整。
